# Issaquah Swingers — 2026 Fantasy Baseball Draft Analysis

A static web app for post-draft analysis of the Issaquah Swingers fantasy baseball league. Includes team grades, season predictions (500 Monte Carlo simulations), year-over-year draft insights, and 2025 standings cross-reference.

## Live Site

Deploy to GitHub Pages:
1. Push this repo to GitHub
2. Go to **Settings → Pages → Source → main branch / root**
3. Your site will be live at `https://IsoscelesKr4mer.github.io/issyfantasybaseball2026/`

## Files

- `index.html` — Page structure
- `styles.css` — All styling (dark theme)
- `data.js` — League data constants (teams, grades, projections, repeat picks, schedules)
- `app.js` — Render functions and initialization
